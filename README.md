## Futtatás

- Először futtasd a backend projekteket, a megfelelő sorrendben

  - API Gateway, Discovery Server
  - Szolgáltatások

- Majd a frontend projekt:

```bash
npm run dev
# vagy
yarn dev
# vagy
pnpm dev
# vagy
bun dev
```

Nyisd meg [http://localhost:3000](http://localhost:3000) böngészőben.
